# ⏭️ Remède Next

The **Remède Next Program** is a program whose goal is to make Remède more scalable and maintainable. It brings a new document schema, more reliable, and a modular architecture.

### Breaking changes

* New database schema
* New Remède document schema
* No more JSON files
